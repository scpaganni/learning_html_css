Modelos de Caixas (Box Model)

Todo elemento visível na tela, dentro de um site, é uma caixa. Uma caixa tem uma altura chamada de height e uma largura chamada de width

Em volta da caixa nós temos aquilo que chamamos de border

Por padrão a linha da border vai ficar grudada ao conteúdo. É possível criar um espaço entre o conteúdo e a border, a esse espaço chamamos padding

É possível também criar um espaço externo, a esse damos o nome de margin.

Tipos de Caixas

Temos dois tipos de caixas: box-level e inline-level

Exemplos de tags box-level:

  • div
  • h1-h6
  • p
  • main
  • header
  • nav
  • article
  • aside
  • footer
  • form
  • video

Exemplos de tags inline-level

  • span
  • a
  • code
  • small
  • strong
  • em
  • sub-sup
  • label
  • button
  • input
  • select
